## Risk Assessment

## Security Considerations
- BCrypt work factor >= 12 in DbSeeder
- Jwt:Key >= 32 chars (256 bits) in appsettings — override via env var `Jwt__Key` in production
- wwwroot/uploads/ does NOT expose directory listing (ASP.NET UseStaticFiles default behavior is safe)
- Never commit production DB password to git — use environment variable override
- CORS: explicitly list only `http://localhost:5173` in Development, no AllowAnyOrigin
